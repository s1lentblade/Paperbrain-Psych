# Orthographic processing in visual word recognition: A multiple read-out model.

**Authors:** Jonathan Grainger, Arthur M. Jacobs
**Year:** 1996
**Citations:** 911
**DOI:** https://doi.org/10.1037/0033-295x.103.3.518

## Abstract

A model of orthographic processing is described that postulates read-out from different information dimensions, determined by variable response criteria set on these dimensions. Performance in a perceptual identification task is simulated as the percentage of trials on which a noisy criterion set on the dimension of single word detector activity is reached. Two additional criteria set on the dimensions of total lexical activity and time from stimulus onset are hypothesized to be operational in the lexical decision task.
