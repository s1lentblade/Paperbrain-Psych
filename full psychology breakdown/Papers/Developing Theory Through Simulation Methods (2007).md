# Developing Theory Through Simulation Methods

**Authors:** Jason P. Davis, Christopher B. Bingham
**Year:** 2007
**Citations:** 1,136
**DOI:** https://doi.org/10.5465/amr.2007.24351453

## Abstract

We describe when and how to use simulation methods in theory development. We develop a roadmap that describes theory development using simulation and position simulation in the “sweet spot” between theory-creating methods, such as multiple case inductive studies and formal modeling, and theory-testing methods. Simulation strengths include internal validity and facility with longitudinal, nonlinear, and process phenomena. Simulation's primary value occurs in creative experimentation to produce novel theory. We conclude with evaluation guidelines.
