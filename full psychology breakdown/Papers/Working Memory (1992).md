## Abstract

The term working memory refers to a brain system that provides temporary storage and manipulation of the information necessary for such complex cognitive tasks as language comprehension, learning, and reasoning. This definition has evolved from the concept of a unitary short-term memory system. Working memory has been found to require the simultaneous storage and processing of information.
