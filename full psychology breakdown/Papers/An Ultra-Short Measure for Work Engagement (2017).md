## Abstract

Abstract. The current study introduces an ultra-short, 3-item version of the Utrecht Work Engagement Scale. Using five national samples from Finland ( N = 22,117), Japan ( N = 1,968), the Netherlands ( N = 38,278), Belgium/Flanders ( N = 5,062), and Spain ( N = 10,040) its internal consistency and factorial validity vis-à-vis validated measures of burnout, workaholism, and job boredom are demonstrated.
