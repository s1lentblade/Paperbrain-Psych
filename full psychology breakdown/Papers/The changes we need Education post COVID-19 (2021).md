## Abstract

The COVID-19 pandemic has caused both unprecendented disruptions and massive changes to education. However, as schools return, these changes may disappear. Moreover, not all of the changes are necessarily the changes we want in education. In this paper, we argue that the pandemic has created a unique opportunity for educational changes that have been proposed before COVID-19 but were never fully realized.
