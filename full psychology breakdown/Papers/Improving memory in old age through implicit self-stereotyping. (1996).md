# Improving memory in old age through implicit self-stereotyping.

**Year:** 1996

## Abstract

This research demonstrates that subliminally activated stereotypes can alter judgments about oneself and can change cognitive performance. In the first study, an intervention that activated positive stereotypes of aging without the participants' awareness tended to improve memory performance, memory self-efficacy, and views of aging in old individuals; in contrast, an intervention that activated negative stereotypes of aging tended to worsen memory performance, memory self-efficacy, and views of aging in old participants.
