**Authors:** Neil Burgess, Graham J. Hitch
**Year:** 1999
**Citations:** 803
**DOI:** https://doi.org/10.1037/0033-295x.106.3.551

## Abstract

A connectionist model of human short-term memory is presented that extends the &amp;apos;phonological loop&amp;apos; (A.D. Baddeley, 1986) to encompass serial order and learning. Psychological and neuropsychological data motivate separate layers of lexical, timing and input and output phonemic information. Connection weights between layers show Hebbian learning and decay over short and long time scales. At recall, the timing signal is rerun, phonemic information feeds back from output to input and lexical nodes compete to be selected. The selected node then receives decaying inhibition.
