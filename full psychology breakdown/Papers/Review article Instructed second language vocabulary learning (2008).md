## Abstract

This article overviews current research on second language vocabulary learning. It concludes that a large vocabulary is necessary to function in English: 8000—9000 word families for reading, and perhaps as many as 5000—7000 families for oral discourse. In addition, a number of word knowledge aspects need to be learned about each lexical item. Taken together, this amounts to a substantial lexical learning challenge, one which many/most learners fail to meet.
