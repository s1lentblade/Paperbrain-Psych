# Automatically Assessing Lexical Sophistication: Indices, Tools, Findings, and Application

**Authors:** Kristopher Kyle, Scott A. Crossley
**Year:** 2014
**Citations:** 516
**DOI:** https://doi.org/10.1002/tesq.194

## Abstract

This study explores the construct of lexical sophistication and its applications for measuring second language lexical and speaking proficiency. In doing so, the study introduces the Tool for the Automatic Analysis of LE xical Sophistication ( TAALES ), which calculates text scores for 135 classic and newly developed lexical indices related to word frequency, range, bigram and trigram frequency, academic language, and psycholinguistic word information. TAALES is freely available; runs on Windows, Mac, and Linux operating systems; and has a simple graphic user interface that allows for batch processing of .txt files.
