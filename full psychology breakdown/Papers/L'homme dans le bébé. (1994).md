# L'homme dans le bébé.

**Authors:** Serge Lebovici
**Year:** 1994
**Citations:** 62
**DOI:** https://doi.org/10.3917/rfp.g1994.58n3.0661

## Abstract

Résumé La transmission intergénérationnelle permet de comprendre les effets des conflits infantiles des parents sur le développement des enfants, mais on doit en saisir les effets dans le processus de subjectivation qu’étaye un véritable maillage narcissique : il s’agit là d’une proposition qui permet, selon l’auteur, d’espérer que l’action thérapeutique s’exerce sur l’homme dans le bébé.
