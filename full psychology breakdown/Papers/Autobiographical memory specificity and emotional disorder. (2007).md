# Autobiographical memory specificity and emotional disorder.

**Authors:** James Williams, Tim Dalgleish
**Year:** 2007
**Citations:** 1,684
**DOI:** https://doi.org/10.1037/0033-2909.133.1.122

## Abstract

The authors review research showing that when recalling autobiographical events, many emotionally disturbed patients summarize categories of events rather than retrieving a single episode. The mechanisms underlying such overgeneral memory are examined, with a focus on M. A. Conway and C. W. Pleydell-Pearce's (2000) hierarchical search model of personal event retrieval. An elaboration of this model is proposed to account for overgeneral memory, focusing on how memory search can be affected by (a) capture and rumination processes, when mnemonic information used in retrieval activates ruminative thinking; (b) functional avoidance, when episodic material threatens to cause affective disturbance;
