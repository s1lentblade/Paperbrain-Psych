## Abstract

The authors summarize developments in the concept of working memory as a multicomponent system, beginning by contrasting this approach with alternative uses of the term working memory. According to a 3-component model, working memory comprises a phonological loop for manipulating and storing speech-based information and a visuospatial sketchpad that performs a similar function for visual and spatial information. Both are supervised by a central executive, which functions as an attentional control system.
