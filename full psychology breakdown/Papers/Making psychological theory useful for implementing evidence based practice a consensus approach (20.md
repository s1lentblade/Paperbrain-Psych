# Making psychological theory useful for implementing evidence based practice: a consensus approach

**Authors:** Susan Michie
**Year:** 2005
**Citations:** 3,560
**DOI:** https://doi.org/10.1136/qshc.2004.011155

## Abstract

A set of behaviour change domains agreed by a consensus of experts is available for use in implementation research. Applications of this domain list will enhance understanding of the behaviour change processes inherent in implementation of evidence-based practice and will also test the validity of these proposed domains.
