## Abstract

In this article, we discuss cognitive load measurement techniques with regard to their contribu-tion to cognitive load theory (CLT). CLT is concerned with the design of instructional methods that efficiently use people’s limited cognitive processing capacity to apply acquired knowledge and skills to new situations (i.e., transfer). CLT is based on a cognitive architecture that consists of a limited working memory with partly independent processing units for visual and auditory information, which interacts with an unlimited long-term memory.
