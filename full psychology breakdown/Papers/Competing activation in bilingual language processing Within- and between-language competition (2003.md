# Competing activation in bilingual language processing: Within- and between-language competition

**Authors:** Viorica Marian, Michael J. Spivey
**Year:** 2003
**Citations:** 643
**DOI:** https://doi.org/10.1017/s1366728903001068

## Abstract

Two eye-tracking experiments examined spoken language processing in Russian-English bilinguals. The proportion of looks to objects whose names were phonologically similar to the name of a target object in either the same language (within-language competition), the other language (between-language competition), or both languages at the same time (simultaneous competition) was compared to the proportion of looks in a control condition in which no objects overlapped phonologically with the target.
