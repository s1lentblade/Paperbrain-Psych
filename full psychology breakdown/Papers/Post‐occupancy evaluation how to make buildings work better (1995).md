## Abstract

Post‐occupancy evaluation (POE) is a diagnostic tool and system which allows facility managers to identify and evaluate critical aspects of building performance systematically. This system has been applied to identify problem areas in existing buildings, to test new building prototypes and to develop design guidance and criteria for future facilities. Outlines the numerous benefits of POE, including better space utilization, as well as cost and time savings. Describes a conceptual framework and evaluation data‐gathering techniques.
