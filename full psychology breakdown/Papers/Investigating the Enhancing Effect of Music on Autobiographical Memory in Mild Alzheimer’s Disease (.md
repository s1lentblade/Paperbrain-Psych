# Investigating the Enhancing Effect of Music on Autobiographical Memory in Mild Alzheimer’s Disease

**Authors:** Muireann Irish, Robert F. Coen
**Year:** 2006
**Citations:** 275
**DOI:** https://doi.org/10.1159/000093487

## Abstract

The enhancing effect of music on autobiographical memory recall in mild Alzheimer's disease individuals (n = 10; Mini-Mental State Examination score >17/30) and healthy elderly matched individuals (n = 10; Mini-Mental State Examination score 25-30) was investigated. Using a repeated-measures design, each participant was seen on two occasions: once in music condition (Vivaldi's 'Spring' movement from 'The Four Seasons') and once in silence condition, with order counterbalanced. Considerable improvement was found for Alzheimer individuals' recall on the Autobiographical Memory Interview in the music condition, with an interaction for condition by group (p < 0.005).
