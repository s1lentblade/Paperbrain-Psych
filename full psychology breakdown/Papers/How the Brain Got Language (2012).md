## Abstract

Unlike any other species, humans can learn and use language. This book explains how the brain evolved to make language possible, through what the text calls the Mirror System Hypothesis. Because of mirror neurons, monkeys, chimps, and humans can learn by imitation, but only "complex imitation," which humans exhibit, is powerful enough to support the breakthrough to language. This theory provides a path from the openness of manual gesture, which we share with nonhuman primates, through the complex imitation of manual skills, pantomime, protosign (communication based on conventionalized manual gestures), and finally to protospeech.
