# Multidirectional Memory: Remembering the Holocaust in the Age of Decolonization

**Authors:** Michael Rothberg
**Year:** 2009
**Citations:** 1,852
**DOI:** https://doi.org/10.1515/9780804783330

## Abstract

Multidirectional Memory brings together Holocaust studies and postcolonial studies for the first time. Employing a comparative and interdisciplinary approach, the book makes a twofold argument about Holocaust memory in a global age by situating it in the unexpected context of decolonization. On the one hand, it demonstrates how the Holocaust has enabled the articulation of other histories of victimization at the same time that it has been declared unique among human-perpetrated horrors. On the other, it uncovers the more surprising and seldom acknowledged fact that public memory of the Holocaust emerged in part thanks to postwar events that seem at first to have little to do with it.
